## Overview
The Sakshi-Protocol is a dual-stream cognitive architecture designed to mitigate stochastic hallucination in Large Language Models (LLMs). It introduces a structural decoupling between task execution and logical oversight.

## Key Components
* **Executive Stream (Manas):** The active generative layer.
* **Supervisory Stream (Sakshi):** The non-participatory witness layer.
* **Mauna Logic Gate:** The high-pass filter for "Strategic Silence."
